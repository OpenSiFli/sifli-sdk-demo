# BLE 对讲机 Demo (talkback)

## SiFli-SDK版本
选择使用SDK main 分支

## 支持的开发板
* sf32lb52-lchspi-ulp

## 概述
核心特性：
- **多人对讲**：支持最多 8 台设备组网，最多 3 人同时讲话
- **自动回连**：Slave 退出房间后可一键回连
- **Opus 音频编码**：使用 Opus 编解码器进行低延迟语音压缩
- **电源管理**：待机 60 秒无操作自动进入低功耗休眠，长按电源键关机/开机
- **电池监测**：实时监测电池电压
- **RGB LED 状态指示**：通过RGB LED 指示当前设备状态

## 设备状态与 LED 指示

| 状态 | Master（蓝色） | Slave（红色） | 说明 |
|------|---------------|--------------|------|
| **待机** | 蓝色常亮 | 红色常亮 | 等待用户操作 |
| **配对中** | 蓝色闪烁 | 红色闪烁 | 正在搜索/等待对方 |
| **等待对讲** | 蓝色闪烁 | 红色闪烁 | 配对完成，等待 Master 确认 |
| **对讲中** | 绿色常亮 | 绿色常亮 | 可进行 PTT 对讲 |
| **接收中** | 橙色常亮 | 橙色常亮 | 正在接收对方语音流 |
| **休眠** | 灭灯 | 灭灯 | 待机超时自动进入低功耗 |

## 按键操作

### KEY1 — 退出/回连

| 当前状态 | 操作 | 效果 |
|---------|------|------|
| 对讲中 | 短按 | 退出当前房间，回到待机 |
| 待机（有回连记录） | 短按 | Slave 发起回连，重新加入上次的房间 |
| 待机 | 短按 | 重置休眠倒计时 |

### KEY2 — 角色切换/配对/PTT

| 当前状态 | 操作 | 效果 |
|---------|------|------|
| 待机 | **短按** | 切换 Master ↔ Slave 角色 |
| 待机 | **长按** | 进入配对模式（Master 创建房间 / Slave 扫描房间） |
| 等待对讲 | **长按** | Master 确认开始对讲 |
| 对讲中 | **按住** | PTT 发言（松开停止） |

## 使用流程

### 1. 编译与烧录

```bash
# 进入项目目录
cd example/ble/talkback/project

# 编译（以 sf32lb52-lchspi-ulp 板型为例）
scons --board=sf32lb52-lchspi-ulp -j8

# 烧录到开发板
build_sf32lb52-lchspi-ulp_hcpu\uart_download.bat

#回车后输入COM端口号
please input the serial port num:COMX

```

### 2. 组网对讲操作

以 2 台设备为例：

#### 步骤一：设置角色
1. **设备 A**：上电默认为 Slave（红灯常亮），**短按 KEY2** 切换为 Master（蓝灯常亮）
2. **设备 B**：保持 Slave 角色（红灯常亮）

#### 步骤二：配对组网
1. **设备 A（Master）**：**长按 KEY2** → 创建房间（蓝灯闪烁）
2. **设备 B（Slave）**：**长按 KEY2** → 扫描并自动加入房间（红灯闪烁）
3. 配对成功后双方进入「等待对讲」状态

> 配对超时时间为 30 秒，超时后自动回到待机

#### 步骤三：开始对讲
1. **设备 A（Master）**：**长按 KEY2** → 确认开始对讲
2. 双方进入「对讲中」状态（绿灯常亮）

#### 步骤四：PTT 对讲
- 任意设备 **按住 KEY2** → 开始讲话（对方自动接收播放）
- **松开 KEY2** → 停止讲话

#### 步骤五：退出房间
- **短按 KEY1** → 退出房间，回到待机
- Slave 退出后可 **短按 KEY1** 一键回连

### 3. 多人组网

支持最多 8 台设备加入同一房间：
1. 1 台 Master 创建房间
2. 多台 Slave 依次长按 KEY2 加入
3. Master 长按 KEY2 确认对讲后，所有成员均可 PTT 通话


## 电源管理

- **自动休眠**：待机状态下 60 秒无按键操作，设备自动进入低功耗模式（关闭 LCD、灭灯、释放 PM_SLEEP_MODE_IDLE）
- **唤醒恢复**：任意按键操作唤醒设备

## 注意事项

1. 需要至少 **2 台设备** 才能完成组网对讲
2. Master 负责创建房间和确认对讲，Slave 负责扫描加入
3. 同一时刻最多 3 人同时讲话
4. 发言中**不可退出房间**，需先松开 KEY2 停止发言
5. 配对超时后自动回到待机，需重新操作
6. 回连功能仅 Slave 可用，且需要之前成功加入过房间
